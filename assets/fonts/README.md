# Шрифты

Сюда положите файлы шрифтов, которые перечислены в `config.BUILTIN_FONTS`:

| Стиль          | Имя файла                  | Где взять (CC/OFL)                                   |
|----------------|----------------------------|------------------------------------------------------|
| Classic        | DejaVuSans.ttf             | системный, обычно уже есть в `/usr/share/fonts/`     |
| Classic Bold   | DejaVuSans-Bold.ttf        | системный                                            |
| Mono           | DejaVuSansMono.ttf         | системный                                            |
| Serif          | DejaVuSerif.ttf            | системный                                            |
| Decorative     | Pacifico.ttf               | https://fonts.google.com/specimen/Pacifico           |
| Minimal        | Montserrat-Regular.ttf     | https://fonts.google.com/specimen/Montserrat         |
| Handwritten    | Caveat.ttf                 | https://fonts.google.com/specimen/Caveat             |
| Gothic         | UnifrakturMaguntia.ttf     | https://fonts.google.com/specimen/UnifrakturMaguntia |
| Futuristic     | Orbitron-Regular.ttf       | https://fonts.google.com/specimen/Orbitron           |
| Cyrillic       | PTSans-Regular.ttf         | https://fonts.google.com/specimen/PT+Sans            |
| Latin Display  | BebasNeue.ttf              | https://fonts.google.com/specimen/Bebas+Neue         |

Если файл какого-то шрифта отсутствует, бот автоматически
подменит его системным `DejaVuSans.ttf`.

Пользователь также может загрузить свой `.ttf` / `.otf` через
кнопку *💧 Водяной знак → ✏️ Шрифт → 📁 TTF/OTF* — файл
будет сохранён в эту папку.
