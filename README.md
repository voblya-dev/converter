# 🎨 Premium Sticker Bot

Telegram-бот для конвертации премиум-стикеров (TGS), обычных стикеров и эмодзи в кастомные видео/изображения с расширенными настройками фона, водяного знака и формата вывода.

## ✨ Возможности

- **Приём контента**: TGS-стикеры (премиум-анимированные), обычные стикеры (статичные/анимированные/видео), текстовые эмодзи
- **Удаление фона**: Автоматическое выделение объекта (стикеры с альфа-каналом обрабатываются нативно; для эмодзи фон отсутствует изначально)
- **Фоны**: 20+ цветов, HEX-ввод, градиенты (горизонтальный/вертикальный/диагональный), пользовательские изображения и видео
- **Форматы вывода**: GIF, MP4, WEBM, PNG
- **Разрешения**: 512×512, 1080×1080, 1920×1080, 1080×1920, 1920×530, кастомное
- **FPS**: 10, 24, 30, 60, "как в исходнике"
- **Качество сжатия**: 4 уровня
- **Водяной знак**: изображение / текст / пресеты, визуальное управление позицией (3×3 сетка + стрелки + смещение по X/Y), прозрачность, размер, поворот, эффекты (тень, обводка, свечение)
- **Шрифты**: 10+ встроенных + загрузка своих TTF/OTF
- **Анимированный прогресс рендеринга** с динамическим прогресс-баром
- **Пресеты, история, мультиязычность (RU/EN)**

## 🚀 Установка

```bash
# 1. Клонировать/распаковать проект
cd sticker_bot

# 2. Установить системные зависимости (Linux)
sudo apt-get update && sudo apt-get install -y ffmpeg libcairo2 libgirepository1.0-dev

# 3. Установить Python-зависимости (Python 3.10+)
pip install -r requirements.txt

# 4. Настроить токен бота
cp .env.example .env
# Открыть .env и вставить токен от @BotFather в BOT_TOKEN

# 5. Запустить
python main.py
```

## 🛠 Технологии

- **aiogram 3.x** — асинхронный Telegram Bot API
- **Pillow + numpy** — композитинг изображений
- **rlottie-python / pyrlottie** — рендер Lottie/TGS в кадры
- **ffmpeg-python** — кодирование в GIF/MP4/WEBM
- **moviepy** — fallback для видеообработки
- **emoji + Pilmoji** — рендер цветных эмодзи
